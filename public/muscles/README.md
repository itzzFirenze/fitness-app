# Muscle Group Icons

Drop your PNG images here. Each file should match the muscle group name (lowercase):

| File name       | Used for   |
|----------------|------------|
| `back.png`     | Back       |
| `chest.png`    | Chest      |
| `biceps.png`   | Biceps     |
| `triceps.png`  | Triceps    |
| `shoulders.png`| Shoulders  |
| `arms.png`     | Arms       |
| `legs.png`     | Legs       |
| `core.png`     | Core       |
| `cardio.png`   | Cardio     |
| `rest.png`     | Rest       |

These are served from `/muscles/back.png` etc. at runtime.
Any missing files fall back to the emoji automatically.
